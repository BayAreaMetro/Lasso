lasso.CubeTransit¶

class lasso.CubeTransit(parameters=<lasso.Parameters.Parameters object>)[source]¶

Bases: object

Class for storing information about transit defined in Cube line files. Has the capability to: - Parse cube line file properties and shapes into python dictionaries - Compare line files and represent changes as Project Card dictionaries
